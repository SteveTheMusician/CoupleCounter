## Setup
- node version: 24.11.1
- npm i

## Run
npm run start

## Build
npm run build

## Some hints

install project in your existing folder
```
npx create-react-app .
```
create new

```
npx create-react-app <ProjectName>
```

fix npm script not found

```
npm install react-scripts --save-dev
```
